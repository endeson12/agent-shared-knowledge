## 1. Identidade Visual (M.E.L)

### Paleta de Cores
```
Primária:    #2E5C8A (Azul serenity — confiança, espiritualidade)
Secundária:  #F4A261 (Laranja warm — alegria, energia infantil)
Destaque:    #E9C46A (Amarelo sol — esperança, fé)
Neutro:      #F8F6F3 (Off-white — limpo, acolhedor)
Texto:       #2C3E50 (Cinza escuro — legibilidade)
```

### Tipografia
- **Títulos:** Poppins Bold (amigável, moderna)
- **Corpo:** Inter Regular (legível, clean)
- **Destaques:** Poppins SemiBold

### Estilo Visual
- Ilustrações flat design com traços suaves
- Personagens diversos e inclusivos
- Elementos geométricos orgânicos (círculos, ondas)
- Espaço em branco generoso (respiração visual)

## 3. Calendário Editorial — Semana 1 (Base)

### Segunda-feira — Educação
**Tema:** "3 Hábitos para Cultivar a Fé nas Crianças"
- **Formato:** Carrossel (5 slides)
- **Copy A.N.A:** Hook + 3 dicas práticas + CTA
- **Design M.E.L:** Template com ícones, cores da paleta, espaço para texto

### Terça-feira — Devocional
**Tema:** "Oração da Noite com os Pequenos"
- **Formato:** Reel (15-30s)
- **Copy A.N.A:** Roteiro com legenda + áudio
- **Design M.E.L:** Animação simples ou vídeo com overlay de texto

### Quarta-feira — Educação
**Tema:** "Como Ensar os 10 Mandamentos de Forma Lúdica"
- **Formato:** Carrossel (6 slides)
- **Copy A.N.A:** Explicação didática + atividade proposta
- **Design M.E.L:** Cards visuais, memória game style

### Quinta-feira — Comunidade
**Tema:** "Qual a maior dificuldade em ensinar sobre Deus?"
- **Formato:** Stories interativo (enquete)
- **Copy A.N.A:** Pergunta engajadora + respostas preparadas
- **Design M.E.L:** Stories template com enquete

### Sexta-feira — Produto
**Tema:** "Kit Introdução à Fé — Lançamento"
- **Formato:** Carrossel (4 slides) + Stories sequência
- **Copy A.N.A:** Storytelling da necessidade + apresentação do kit + oferta
- **Design M.E.L:** Mockup do produto, depoimentos, preço, CTA

## 4. Templates de Design (M.E.L)

### Carrossel Educativo (5 slides)
1. **Capa:** Hook visual + título grande
2. **Dica 1:** Ícone + texto curto
3. **Dica 2:** Ícone + texto curto
4. **Dica 3:** Ícone + texto curto
5. **CTA:** "Salve para consultar" + @pequenoscomdeus

### Reel Devocional (15-30s)
- Segundos 0-3: Hook visual
- Segundos 3-20: Conteúdo principal
- Segundos 20-25: Reflexão
- Segundos 25-30: CTA

### Stories Interativo
- Fundo da paleta
- Pergunta em destaque
- Opções de enquete
- Logo discreta
