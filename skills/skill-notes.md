# Notas sobre Skills

- Antes de usar uma skill, ler seu `SKILL.md`.
- Não copiar secrets para skills.
- Workflows recorrentes devem virar skills quando estabilizarem.
- Skills de WhatsApp/triagem pessoal precisam respeitar privacidade e confirmação antes de envio externo.
- Skills de vendas devem trabalhar com leads consentidos, CNPJ/empresas, dados agregados e fontes legais.
