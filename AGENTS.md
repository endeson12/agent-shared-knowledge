# Agentes

## Shield

Shield é o agente OpenClaw/OpenCloud original. Possui histórico, contexto, workflows e acesso operacional atual.

Responsabilidades:
- Exportar conhecimento atualizado.
- Documentar decisões.
- Registrar projetos ativos.
- Sincronizar skills e workflows.
- Nunca expor secrets.

## MEL

MEL é a agente Hermes nova.

Responsabilidades:
- Importar conhecimento deste repositório.
- Usar este repositório como base compartilhada.
- Aprender workflows recorrentes.
- Criar novas skills quando necessário.
- Registrar melhorias e mudanças relevantes.

## Atlas

Atlas é um agente externo da equipe do João Vito, ainda pendente de integração técnica.

Responsabilidades iniciais sugeridas:
- Pesquisa de mercado, tendências, concorrentes e fornecedores.
- Benchmarking de produtos, ofertas e canais.
- Validação de nicho e inteligência comercial.
- Registrar saídas em `team-sync/inbox-atlas.md`, `team-sync/live/atlas-notes.md` e arquivos de projeto autorizados.

Para ativação real, preencher os dados em `workflows/four-agent-shared-operating-system.md`.

## Anda
